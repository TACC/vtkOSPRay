Projects using / related to vtkOSPRay

This page gives a brief (and incomplete) list of other projects that make use of vtkOSPRay, as well as a set of related links to other projects and related information.

If you have a project that makes use of vtkOSPRay and would like this to be listed here, please let us know.

Projects that use vtkOSPRay

"ParaView OSPRay Plugin"

The ParaView OSPRay Plugin (http://tacc.github.io/pvOSPRay/) is based on the vtkOSPRay library.

"VisIt OSPRay Plugin"

The VisIt OSPRay Plugin (http://tacc.github.io/visitOSPRay/) is based on the vtkOSPRay library.

Projects that vtkOSPRay makes use of

  • The OSPRay Ray Tracer (ospray.github.io)
  • The Embree Ray Tracing Kernel Framework (embree.github.io)
  • The Intel® SPMD Program compiler (ISPC, ispc.github.io)
  • OSPRay's "sister project" is the OpenSWR software rasterizer, a new high-performance, highly-scalable, software OpenGL implementation that runs on host CPUs.
http://tacc.github.io/vtkOSPRay/
